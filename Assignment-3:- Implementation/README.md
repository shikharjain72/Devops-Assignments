1. Open terminal in project folder:
   ```bash
   cd "/Users/shikharjain/Desktop/Devops/ALL-PROJECTS/Devops-Assignments/Assignment-3:- Implementation"
   ```

2. (Optional but recommended) Create a virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

3. Install dependencies:
   ```bash
   python3 -m pip install -r requirements.txt
   ```

4. Start the Flask app:
   ```bash
   python3 app.py
   ```

5. Open in browser:
   - `http://localhost:8080`

## Stop the App

- Press `Ctrl + C` in terminal.

## Notes

- The app is configured to run on `0.0.0.0:8080`.
- Update `index.html` to customize portfolio content.
